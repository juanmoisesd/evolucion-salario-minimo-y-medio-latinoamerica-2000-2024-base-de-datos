## Descripción

Base de datos sobre la evolución del salario mínimo y salario medio en los países de Latinoamérica para el período 2000-2024. Permite comparar la progresión salarial entre países y a lo largo del tiempo, incluyendo datos en moneda local y equivalentes en USD.
